# merge-sort

[16,21,11,8,12,22] -> Merge Sort

Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
Big-O gösterimini yazınız.

 
 1)          |16|21|11|8|12|22|            ->n
        
  |16|21|11|              |8|12|22|        ->n
    
  |16|21|  |11|           |8|12|  |22|     ->n
       
  |16| |21| |11|          |8| |12| |22|    ->n
  
  |16|21|  |11|           |8|12|   |22|    ->n
 
  |11|16|21|              |8|12|22|        ->n
        
            |8|11|12|16|22|                ->n



2) Big-O notation: O(nlogn)
